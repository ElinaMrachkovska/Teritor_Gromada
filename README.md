# Кам'янець-Подільська громада — офіційний сайт (React + Tailwind)

Повноцінний React-проєкт на Vite з Tailwind CSS v4 та маршрутизацією через
react-router-dom. Структура та дизайн повторюють референс hromady.org,
контент — про територіальну громаду міста Кам'янця-Подільського.

## Стек

- React 19
- Vite 8
- Tailwind CSS v4 (через `@tailwindcss/postcss`)
- react-router-dom v7

## Запуск

```bash
npm install
npm run dev
```

Сайт відкриється на http://localhost:5173

## Збірка для продакшну

```bash
npm run build
```

Готові файли з'являться в папці `dist/`. Її можна задеплоїти на будь-який
статичний хостинг (Netlify, Vercel, GitHub Pages тощо).

## Структура проєкту

```
src/
  components/      # Header, Footer, Layout, PageHero, NewsCard, DocTable
  pages/            # Home, News, DocumentsLegal, Gallery, Council,
                     # DocumentsOfficial, Structure, Contacts
  data/
    content.js      # усі тексти, новини, документи, люди — редагуй тут
  index.css         # Tailwind + кастомна тема (кольори, шрифти)
```

## Маршрути

| Шлях                  | Сторінка                          |
|------------------------|------------------------------------|
| `/`                     | Головна                            |
| `/news`                 | Новини                             |
| `/documents-legal`      | Законодавчі документи              |
| `/gallery`               | Галерея                            |
| `/council`               | Рада громади                       |
| `/documents-official`   | Офіційні документи                 |
| `/structure`             | Управлінська структура             |
| `/contacts`               | Контакти (форма звернення)         |

## Як редагувати контент

Більшість текстового контенту (новини, документи, склад ради, структурні
підрозділи, фото галереї) зберігається в `src/data/content.js` — зміни там
автоматично відобразяться на відповідних сторінках.

Кольори та шрифти налаштовані в `src/index.css` у блоці `@theme`:

- `--color-ink` — основний темно-синій
- `--color-ochre` — акцентний колір (вохра)
- `--color-stone` / `--color-paper` — фонові кольори
- `--font-display` (Lora) — заголовки
- `--font-body` (Inter) — основний текст
